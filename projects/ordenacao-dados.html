Pesquisa e Ordenação de Dados

01 - 05 -2019
Lucas Lima

Métodos de pesquisa e ordenação são constantemente estudados e aprimorados para serem aplicados em diferentes aplicações no mercado de TI. Com isso temos aprimorado técnicas para classificar e ordenar elementos com algoritmos menos complexos e mais eficazes, capazes de organizar milhares de dados em apenas alguns segundos que em tempos de processamento em nuvem precisamos de requisições e respostas cada vez mais rápidas.

Os algoritmos de ordenação ou classificação são estudados e desenvolvidos a muitos anos desde o início da computação, sendo ate hoje os mais usados: Bubble sort, Isertion sort, Comb sort e Shell sort. Cada um apresenta uma peculiaridade podendo ser referente a sua complexidade logarítmica, quantidade de intervalos ou os chamados Gaps e a sequência de ordenação dos dados a quais são indicados, para se escolher qual algoritmo se encaixa melhor para sua aplicação deve se analisar esses três requisitos que acabei de citar.

O projeto que trago aqui é uma pesquisa e explicação detalhada de como funciona quatro desses algoritmos:

*Bubble sort
*Isertion sort
*Comb sort
*Shell sort

O foco é mostrar uma determinada quantidade de dados que estão sendo manipulados, o valor deles e a quantidade de trocas para se organizar esses dados de diferentes formas.
